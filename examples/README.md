Welcome to the Pumpkin examples! Here you will find simple programs to help you learn.

## How to Run

You can run any example using the terminal:

```bash
node --loader ts-node/esm src/run.ts examples/filename.pumpkin
```

## The Examples

1. **[hello_world.pumpkin](hello_world.pumpkin)**: The most basic program. Prints a message.
2. **[ask_name.pumpkin](ask_name.pumpkin)**: Shows how to get input from the user.
3. **[calculator.pumpkin](calculator.pumpkin)**: Demonstrates variables and math.
4. **[age_checker.pumpkin](age_checker.pumpkin)**: Teaches `if` and `else` logic.
5. **[countdown.pumpkin](countdown.pumpkin)**: Uses a `repeat` loop.
6. **[shopping_list.pumpkin](shopping_list.pumpkin)**: Shows how to use lists.
7. **[guess_number.pumpkin](guess_number.pumpkin)**: A simple game using a `while` loop.
8. **[daily_routine.pumpkin](daily_routine.pumpkin)**: Explains how to create and use `function`s.
