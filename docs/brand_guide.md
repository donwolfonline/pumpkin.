## 1. Brand Story

**Why Pumpkin exists:**
Programming has become exclusive. It's full of jargon, cryptic errors, and complex setups that scare away curious minds. We believe that coding is a creative superpower that should be accessible to everyoneâ€”not just engineers.

**The Problem:**
"I want to learn to code, but it looks too hard."

**The Solution:**
Pumpkin is a language designed to be *read*. It removes the scary parts (semicolons, braces, cryptic errors) and replaces them with friendly, human-readable instructions.

## 4. Visual Identity

### Color Palette

* **Pumpkin Orange** (`#FF7518`): Primary brand color. Warm and energetic.
* **Vine Green** (`#3B8132`): Success messages, growth, correct code.
* **Night Sky** (`#1A1A2E`): Backgrounds for code snippets. High contrast.
* **Starlight** (`#FEF9E7`): Text color. Soft on the eyes.

### Typography

* **Headings**: Clean, rounded sans-serif (e.g., *Nunito* or *Quicksand*).
* **Body**: Readable sans-serif (e.g., *Inter* or *Open Sans*).
* **Code**: Monospace with character (e.g., *Fira Code* or *JetBrains Mono*).

## 5. Public Messaging

### Homepage Hero

**Headline**: The Programming Language You Can Read.
**Subhead**: Stop wrestling with syntax. Start building ideas. Pumpkin is the beginner-friendly language that teaches you as you type.

### "Why Pumpkin?" (Comparison)

* **Python**: Great, but indentation errors are tricky for beginners.
* **JavaScript**: Powerful, but too many confusing symbols (`{}`, `[]`, `=>`, `;`).
* **Pumpkin**: No indentation rules. English keywords. Helpful errors.
