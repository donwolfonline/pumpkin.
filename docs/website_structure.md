## 3. Recommended Navigation Flow

**The "Happy Path" for a Beginner:**

1. **Home**: Reads "Code for Humans". Sees cute pumpkin.
2. **Home**: Clicks "Try Pumpkin Now".
3. **Playground**: Runs "Hello World". Feels successful.
4. **Playground**: Wants to do more -> Clicks "Learn".
5. **Learn**: Starts "Level 1" of the curriculum.

**The "Dev Path":**

1. **Home**: Scans comparison code.
2. **Docs**: Checks syntax diagrams and standard lib.
3. **GitHub**: Checks source code and instructions.
