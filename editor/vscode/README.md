# Pumpkin Language Support

Official syntax highlighting and file icons for the Pumpkin programming language.

## Features

- **File Icon**: Beautiful orange pumpkin icon for `.pumpkin` files.
- **Syntax Highlighting**: Professional color-coding for keywords, strings, and comments.

## Installation

### From Marketplace

Search for "Pumpkin Language Support" in the VS Code Extensions view.

### Manual Installation

1. Download the `pumpkin-vscode-0.1.0.vsix` file.
2. Open VS Code.
3. Go to the Extensions view (Ctrl+Shift+X).
4. Click the "..." (Views and More Actions) menu and select "Install from VSIX...".
